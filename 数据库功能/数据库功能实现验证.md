# 数据库功能实现验证

本验证文档用于在本地 PostgreSQL16 环境中，对“翼型工程数据库系统”的数据库侧功能做可重复的验收检查。当前已将数据迁移并统一到 **airfoil_db**，因此本文以 airfoil_db 为主进行验收与导出说明。

- **airfoil_db**：按《数据库功能技术方案.md》落地的一套独立 schema（已完成从 public 的数据迁移）。
- **public + api**：项目既有数据模型与查询函数（作为迁移源与对照，必要时用于回归比对）。

若你只关心统一后的功能验收与导出，请直接使用 **airfoil_db** 的表/视图/函数。

## 0. 连接方式

本仓库提供了便捷脚本用于执行只读检查 SQL：

- [check_airfoil_db.ps1](file:///d:/%E9%9D%A2%E5%90%91%E6%95%B0%E6%8D%AE%E6%99%BA%E8%83%BD%E5%8D%8F%E5%90%8C%E7%9A%84%E7%BF%BC%E5%9E%8B%E5%B7%A5%E7%A8%8B%E6%95%B0%E6%8D%AE%E5%BA%93%E7%B3%BB%E7%BB%9F/%E6%95%B0%E6%8D%AE%E5%BA%93%E5%8A%9F%E8%83%BD/check_airfoil_db.ps1)

脚本会读取：

- `.env`：数据库密码
- `数据库设计/sql/pg_service.conf`：host/port/dbname/user

建议执行方式（避免 PowerShell Profile 干扰）：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File "数据库功能\check_airfoil_db.ps1" -Query "select 1;"
```

select count(*) as airfoil_cnt from airfoil_db.airfoil;

## 2. 功能点验收对照（airfoil_db）

### 2.1 翼型基础信息管理

验证：能查询翼型编号、名称、类别、来源等信息。

```sql
select airfoil_code, airfoil_name, category, source, generation_method
from airfoil_db.airfoil
order by airfoil_code
limit 10;
```

### 2.2 几何坐标管理（按翼型读取完整轮廓并区分表面）

使用函数：`airfoil_db.get_airfoil_geometry(airfoil_code, version_no)`

```sql
select airfoil_code, version_no, surface, count(*) as point_cnt
from airfoil_db.get_airfoil_geometry('ag03', null)
group by airfoil_code, version_no, surface
order by surface;
```

期望：

- 同一翼型能返回上/下表面（surface）分组
- 点序（seq）可用于恢复曲线

### 2.3 性能数据管理（多条件查询）

使用函数：`airfoil_db.find_airfoils_by_condition(reynolds, mach, aoa_deg, min_cl, max_cd, min_ldr, only_latest)`

```sql
select *
from airfoil_db.find_airfoils_by_condition(500000, null, -2, null, 0.05, null, true)
order by ldr desc nulls last
limit 10;
```

期望：

- 能按工况筛选并按指标排序输出

### 2.4 数据版本管理（保留历史版本、支持对比）

使用函数：

- `airfoil_db.get_airfoil_performance_across_versions(airfoil_code, reynolds, aoa_deg, mach)`
- `airfoil_db.compare_airfoil_versions(airfoil_code, reynolds, aoa_deg, version_no_a, version_no_b, mach)`

```sql
select *
from airfoil_db.get_airfoil_performance_across_versions('ag03', 500000, 2, null)
order by version_no;

select *
from airfoil_db.compare_airfoil_versions('ag03', 500000, 2, 4, 5, null);
```

期望：

- 同一翼型能返回多个 version_no 的记录
- status 字段可用于区分版本状态（draft/released/archived）

### 2.5 异常数据识别

数据侧异常来源：

- `airfoil_db.anomaly_record`（迁移落表）
- `airfoil_db.fn_detect_anomaly()`（规则检测函数，可再次检测并增量写入）

使用函数：`airfoil_db.list_airfoils_with_anomalies(only_latest)`

```sql
select *
from airfoil_db.list_airfoils_with_anomalies(true)
order by anomaly_count desc, airfoil_code
limit 10;
```

可选：再次运行规则检测（增量写入，不覆盖既有异常记录）：

```sql
select * from airfoil_db.fn_detect_anomaly(null) order by rule_code;
```

### 2.6 查询与分析（曲线对比）

使用函数：`airfoil_db.compare_airfoils_at_reynolds(airfoil_codes[], reynolds, mach, only_latest)`

```sql
select *
from airfoil_db.compare_airfoils_at_reynolds(array['ag03','ag10','ag13'], 500000, null, true)
order by airfoil_code, aoa_deg
limit 30;
```

期望：

- 同一雷诺数下能输出多个翼型的曲线点（aoa_deg, cl, cd, ldr）

## 3. 导出与展示（CSV/曲线）

数据库侧推荐导出“扁平结果”，由应用层负责画图。

### 3.1 直接导出曲线点（适用于绘图）

```sql
select *
from airfoil_db.compare_airfoils_at_reynolds(array['ag03'], 500000, null, true)
order by aoa_deg;
```

### 3.2 使用视图导出（适用于结果批量导出）

airfoil_db 已提供导出视图：

- `airfoil_db.v_export_geometry`
- `airfoil_db.v_export_performance`
- `airfoil_db.v_export_anomaly`
- `airfoil_db.v_export_geometry_latest`
- `airfoil_db.v_export_performance_latest`
- `airfoil_db.v_export_anomaly_latest`

示例（导出某翼型最新版本的几何点）：

```sql
select *
from airfoil_db.v_export_geometry
where airfoil_code = 'ag03'
  and version_no = (select max(version_no) from airfoil_db.airfoil_version v join airfoil_db.airfoil a on a.airfoil_id=v.airfoil_id where a.airfoil_code='ag03')
order by surface, seq;
```

### 3.3 CSV 一键导出（推荐）

使用脚本 [export_airfoil_db.ps1](file:///d:/%E9%9D%A2%E5%90%91%E6%95%B0%E6%8D%AE%E6%99%BA%E8%83%BD%E5%8D%8F%E5%90%8C%E7%9A%84%E7%BF%BC%E5%9E%8B%E5%B7%A5%E7%A8%8B%E6%95%B0%E6%8D%AE%E5%BA%93%E7%B3%BB%E7%BB%9F/%E6%95%B0%E6%8D%AE%E5%BA%93%E5%8A%9F%E8%83%BD/export_airfoil_db.ps1) 执行任意 SELECT 并导出为 CSV（自动读取 `.env` 与 `pg_service.conf`）。

示例（导出 ag03 在 Re=500000 的曲线点到 CSV）：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File "数据库功能\export_airfoil_db.ps1" `
  -OutputPath "数据库功能\export\ag03_re500k_curve.csv" `
  -Query "select * from airfoil_db.compare_airfoils_at_reynolds(array['ag03'], 500000, null::double precision, true) order by aoa_deg"
```

## 4. 迁移说明（public -> airfoil_db）

`airfoil_db` 使用 bigint identity 作为主键，public 使用 uuid 作为主键。迁移过程中会在 airfoil_db 下生成映射表用于追溯：

- `airfoil_db.map_airfoil`（public.airfoil_id -> airfoil_db.airfoil_id）
- `airfoil_db.map_version`（public.version_id -> airfoil_db.version_id）
- `airfoil_db.map_perf`（public.record_id -> airfoil_db.perf_id）
- `airfoil_db.map_rule`（public.rule_id -> airfoil_db.rule_id）
